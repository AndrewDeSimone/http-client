# http-client

A minimal HTTP client implemented in C. Fetches a resource at a given URL using an HTTP GET request and displays the response exactly as received.

## Features

- Fetch resources via HTTP GET
- Display response including headers (optional)
- Command-line interface

## Build Instructions

This project uses GCC and a Makefile for building.

### Requirements

- MinGW (GCC) installed and added to your PATH
- `mingw32-make` installed

### Build

Open a terminal in the project folder and run:

```bash
mingw32-make
